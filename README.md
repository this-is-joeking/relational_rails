# Relational Rails

* Rails Version 5.2.8.1
* Ruby Version 2.7.2

**To Run**
1. Once repo is cloned run a `bundle install`
1. rails db:{drop,create,migrate,seed}
1. rails s
1. open browser to http://localhost:3000/
